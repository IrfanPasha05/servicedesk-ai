# 🚀 ServiceDesk Narrator AI

> AI-powered Service Desk Documentation Assistant that transforms raw incident notes, Teams chats, and troubleshooting details into professional ServiceNow documentation.

🌐 **Live Demo:** https://servicedesk-ai-seven.vercel.app/

---

## 📌 Overview

ServiceDesk Narrator AI is a modern AI-powered web application designed for IT Service Desk Engineers.

Instead of manually writing ServiceNow ticket updates, users can simply paste an incident summary or Microsoft Teams conversation. The AI automatically generates professional documentation while preserving troubleshooting actions, escalation details, and chronological order.

This project was built to improve documentation quality, reduce manual effort, and increase productivity for IT support teams.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/IrfanPasha05/servicedesk-ai.git
```

Go to the project folder

```bash
cd servicedesk-ai
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
